# henri-potier-react

> A little exercice

## Build Setup

```bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn start

# build for production
$ yarn build

# unit tests
$ yarn test

# e2e tests
$ yarn e2e

# generate json version of e2e tests report
$ yarn e2e:json

# generate and display html version of e2e report
$ yarn e2e:report

# linters
$ yarn lint
```
